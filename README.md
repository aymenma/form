# 📋 Interactive Form – React

A responsive interactive form built with React.js featuring real-time input validation.

## ✨ Features

- Real-time field validation
- Age validation (must be between 18 and 100)
- Phone number validation (10, 11 or 12 digits)
- Success or error message on submission
- Clean and responsive UI

## 🛠️ Technologies Used

- React.js
- JavaScript
- HTML & CSS
- Bootstrap

## 🚀 Live Demo

[View Demo](https://form-loan.netlify.app/)

## 📦 Installation

git clone https://github.com/aymenma/form
cd form
npm install
npm start
